# 書籍企画書作成スキル

このスキルは、`docs/ideas/book-concept.md`を元に高品質な書籍企画書を作成するためのガイドです。

## 前提条件

### docs/ideas/book-concept.md が配置されている

書籍コンセプト（壁打ちの成果物）が存在する必要があります。

## 既存ドキュメントの優先順位

1. **既存の書籍企画書 (`docs/book-plan.md`)** - 最優先
2. **このスキルのガイド** - 参考資料

## 出力先

```
docs/book-plan.md
```

## テンプレートの参照

書籍企画書を作成する際は、次のテンプレートを使用してください: ./template.md
