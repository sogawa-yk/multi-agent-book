# AIエージェント開発入門 ― シングルからマルチへ

OCI Generative AI Serviceで学ぶマルチエージェントの設計と実装

## 概要

本リポジトリは、Claude Codeによるスペック駆動開発で技術書を執筆するプロジェクトです。

## 使い方

### 初回セットアップ

```bash
claude
> /setup-book
```

### 章の執筆

```bash
> /write-chapter 4
```

### 章のレビュー

```bash
> /review-chapter 4
```

### フィードバック修正

```bash
> /fix-feedback 2026-03-05-ch04-review.md
```

詳細は `CLAUDE.md` を参照してください。
