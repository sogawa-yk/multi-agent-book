# 第9章 OCI上でシングルエージェントを構築する ― 仕様書

## 章の概要

**章の目的**: OCI GenAI Serviceを使ってエージェントを1つ作り、マルチエージェントシステムの基盤となるシングルエージェントを確立する。

**目標ページ数**: 22p

**所属する部**: 第III部 OCI上での実践

## 前提知識

この章を読むために必要な前提:
- 第2章の内容（ツール、MCP、コンテキスト管理）
- 第8章の内容（OCI GenAI Service、Python SDK）

## 章末の理解度チェック問題

### 問題の方針
- 問題数: 4問
- 問題の種類: 概念の確認、判断問題、設計問題

### 問題案
1. Resource PrincipalとInstance Principalの違いと使い分けを説明せよ（種類: 概念の確認）
2. ReActループの停止条件として適切なものを3つ挙げよ（種類: 概念の確認）
3. OCI APIから429（レート制限）エラーが返された場合の適切な対処を述べよ（種類: 判断問題）
4. ツール実行結果が大きすぎてコンテキストウィンドウに収まらない場合の設計を提案せよ（種類: 設計問題）
